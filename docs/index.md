# Event Aggregator

A Django web application for managing and discovering events.  
This project was developed as part of the CS50W capstone and is now published with reproducible instructions.

---

## 🚀 Features
- Create, update, and delete events
- Browse upcoming events in a clean interface
- User authentication and event ownership
- Responsive design for desktop and mobile

---

## 📦 Installation
Clone the repository and set up the environment:

```bash
git clone https://github.com/intiserIqbal/event-aggregator.git
cd event-aggregator
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

## 📝 Usage
1. Register or log in to your account.
2. Add new events with details like title, date, and description.
3. Browse events created by others.
4. Edit or delete your own events.
