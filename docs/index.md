# Event Aggregator

Welcome to the Event Aggregator project!  
This site provides a quick overview and links to full documentation.

## 📖 Project Overview
Event Aggregator is a Django web app for discovering, creating, and managing events with RSVP tracking and interactive maps.

👉 [View full README on GitHub](https://github.com/intiserIqbal/event-aggregator)

## 🚀 Quick Start
- Clone the repo
- Install dependencies
- Run migrations
- Start server

For detailed instructions, see the README.
